Accessibility Statement for SSLCom

This is an accessibility statement from University of Sheffield (by student of TUOS Myron Sukhanov).

Conformance status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. SSLCom is fully conformant with WCAG 2.1 level AA. Fully conformant means that the content fully conforms to the accessibility standard without any exceptions.

Technical specifications

Accessibility of SSLCom relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • CSS

These technologies are relied upon for conformance with the accessibility standards used.

Assessment approach

University of Sheffield assessed the accessibility of SSLCom by the following approaches:

  • Self-evaluation

Date

This statement was created on 14 November 2023 using the W3C Accessibility Statement Generator Tool.

Accessibility considerations

  • Black text on the white background ensures the highest contrast ratio, therefore, promotes accessibility
  • The website can be read via built in screen readers, including images.
  • Wave Evaluation tool has been used to ensure accessibility.
  • All the text on the website is big enough (12px and larger) and the font makes the text look "bold" to ensure readibility
